S&S FULL-STACK WEBSITE

1. In Supabase SQL Editor, run the complete file: supabase-admin-policies.sql
2. Keep public sign-up disabled in Supabase Authentication if only you should manage the site.
3. Open index.html to view the public website.
4. Open index.html#admin to log in and add sections/products.
5. Upload this whole folder to GitHub, then deploy it on Vercel.

The Supabase publishable key in config.js is safe for browser use because access is controlled by Row Level Security.
Never add a secret/service_role key to this project.
